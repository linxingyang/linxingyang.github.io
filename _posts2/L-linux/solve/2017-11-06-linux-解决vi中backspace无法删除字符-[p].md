转载

最近在学linux的vi编辑器，结果 backspace 不能删老不习惯里。百度了一下，下面这个可以用。

Ubuntu的vi不支持方向键和退格键，所以要想加入这些功能配置如下：

```
vi /etc/vim/vimrc.tiny
将 set compatible 
改为set nocompatible
加入一句：set backspace=2
```
